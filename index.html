Independent security systems
built with real components

JC Laboratories builds independent security systems across encrypted storage, cryptographic platforms, public gateway layers, and governed AI. The products are different. The standard is the same: defined architecture, standard primitives, and controlled execution.

Cryptographic Identity

Cryptaverse is built around a two-anchor-key identity model using Ed25519 for signatures and X25519 for encryption and key exchange.

MessageCrypt Stack

Encrypted messaging is built around ephemeral X25519, HKDF-derived symmetric keys, AES-256-GCM content encryption, and Ed25519 authenticity.

Password Hardening

Argon2id is the target password KDF for vault and Safety Deposit Box protection, with memory-hard derivation designed for stronger offline resistance.

Public Gateway Layer

CipherPort is the public gateway surface of Cryptaverse, built around boards, controlled entry points, and gateway functions rather than the full private system.

EGAF Governance Model

The Entropy-Gated AI Framework is built around OIL_CONTRACT, INVOKER, and E_T for governed, structured, and auditable AI task execution.

Gaffer Runtime

Gaffer is the flagship assistant built on EGAF, delivered as a governed AI interaction system instead of an unconstrained chatbot layer.

Application Stack

Across the portfolio, the stack includes Flutter and Dart on the client side, Cloudflare services for platform infrastructure, and Railway for Gaffer runtime delivery.

Platform Direction

The portfolio combines encrypted vault software, a cryptographic platform architecture, a public gateway layer, and an AI governance framework under one company.

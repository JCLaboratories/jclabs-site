What Aleph Vault Includes

Encrypted Vault

Store notes, credentials, and sensitive data inside a local encrypted vault.

MessageCrypt

Encrypted messaging built around X25519, AES-256-GCM, and Ed25519 signatures.

Encrypted Notes

Markdown and plain text notes stored inside the vault and exportable when needed.

Password Database

Encrypted label/value storage for credentials and other structured secrets.

Key Pair Management

Create and manage cryptographic key pairs for encryption, signing, and identity functions.

Password Generation

Cryptographically secure password generation with configurable character sets and length.

How to Install
1

Download the APK

Tap the Download button above on your Android device, or transfer the file from your computer via USB or cloud storage.

2

Allow Unknown Sources

Go to Settings → Security → Install Unknown Apps. Select your browser or file manager and enable Allow from this source.

3

Open the APK File

Tap the downloaded file in your notifications or file manager. Android will show an install prompt. Tap Install.

4

Create Your Account & Vault

Open Aleph Vault. Create a free account with your email. Then create your vault password — write it down somewhere safe. It cannot be recovered if lost.

5

You're Secured

Your vault is created. A key pair is generated automatically. You can start storing notes, passwords, and sending encrypted messages immediately.
